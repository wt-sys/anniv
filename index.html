Protection des données

Collecte de données personnelles
Cette page collecte les informations suivantes lors de votre visite :
• Votre adresse IP
• Date et heure d'accès
• Identifiant de session

Finalités du traitement
Ces données sont utilisées pour :
• Analyser l'engagement des utilisateurs
• Améliorer l'expérience utilisateur
• Respecter les obligations légales

Base légale (RGPD)
Le traitement de vos données repose sur votre consentement explicite (article 6.1.a du RGPD).

Durée de conservation
Les données sont conservées pendant 90 jours à partir de la date de collecte, après quoi elles sont supprimées automatiquement.

Vos droits
Conformément au RGPD, vous disposez des droits suivants :
• Droit d'accès à vos données
• Droit de rectification
• Droit à l'effacement ("droit à l'oubli")
• Droit de limitation du traitement
• Droit à la portabilité des données
• Droit d'opposition

Pour exercer ces droits, contactez-nous via le formulaire de contact.
